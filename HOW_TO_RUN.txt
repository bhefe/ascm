================================================================================
                    SCAN SOFTWARE - HOW TO RUN
================================================================================

FOR NON-ADMIN USERS (No Administrator Rights):
================================================================================

1. Double-click "Run_Scan_Software.bat" OR "Run_Scan_Software.vbs"

2. A command window will appear showing:
   - Checking for the software
   - Copying to temporary folder
   - Launching

3. Wait for the scanning to complete

4. Excel report will open automatically

Note: This method requires NO admin privileges. The software copies itself
to your temp folder and runs from there, which bypasses most restrictions.


===========================================================================

FOR ADMIN USERS (With Administrator Rights):
================================================================================

Option A: USE INSTALLER (Recommended)
--------------------------------------

1. Right-click "Install_Scan_Software.vbs"

2. Select "Run as administrator"

3. Follow the installation wizard

4. Once installed, find "Scan Software" in your Start Menu


Option B: TEMPORARY FOLDER METHOD
----------------------------------

1. Double-click "Run_Scan_Software.bat" or "Run_Scan_Software.vbs"

2. Software will run from temporary location

3. Excel report opens automatically


Option C: UNBLOCK METHOD
------------------------

1. Right-click "Scan Software.exe"

2. Click "Properties"

3. Check "Unblock" at the bottom (if available)

4. Click "Apply" → "OK"

5. Double-click "Scan Software.exe" to run


===========================================================================

TROUBLESHOOTING:

If you see an error message:
- Try the other launcher (batch vs VBS)
- Contact your IT department if issues persist

The software will create a compliance report in your Downloads folder
when complete.
